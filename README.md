# Web Inventory System

This project was an opportunity for me to implement user interfaces and leverage existing frameworks, assets, and content for **object-oriented programming**.

Through it, I was able to:

* **Customize a Spring Framework application** to track inventory for a hypothetical skateboard shop.
* **Implement inventory validation**, including displaying error messages to the user.
* **Write unit tests** using JUnit to ensure the application's logic for inventory validation is working correctly.


## Basic Instructions

To get the application up and running:

1.  Clone the project.
2.  Open the project in **IntelliJ**.
3.  Run the application.
4.  Navigate to this URL in your browser: `http://localhost:8080/`.


## Usage Instructions

The application's main screen displays all parts and products in the database.

* **Add:** Click the `Add Inhouse Part`, `Add Outsourced Part`, or `Add Product` buttons to add new items to the database.
* **Update:** Click the **Update** button next to any item to modify its details.
* **Delete:** Click the **Delete** button to remove a part or product. 
* **Buy Now:** Click the **Buy Now** button next to a product to decrement its inventory by one.
